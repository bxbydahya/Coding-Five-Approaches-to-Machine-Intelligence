# Coding Five: Approaches to Machine Intelligence

This project contains three connected machine learning tasks built in Python.

## Task 1
Classify a grocery item name into a grocery category.

Examples:
- apple → Fruits & Vegetables
- milk → Eggs & Dairy

## Task 2
Recommend an extra grocery item based on a shopping basket.

Examples:
- pasta + olive oil → canned tomatoes

## Task 3
Classify an image of a grocery item, identify the product, then assign it to the correct grocery category.

Examples:
- image of apple → apple → Fruits & Vegetables

## Tools
- Python
- pandas
- scikit-learn
- mlxtend
- tensorflow
- matplotlib

## Structure
- `src/` contains Python scripts
- `data/` contains datasets
- `models/` contains saved models
- `results/` contains outputs
- `docs/` contains project documentation
